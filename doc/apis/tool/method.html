零碎方法

计算类

color 

var rgbaArray = $$.color(param);

传递一个合法的CSS颜色字符串，统一转换为rgba(x, x, x, x)格式的数据，返回数组[r, g, b, a]。

辅助类

loop 

$$.loop(datas, function(value, key, index){
    // 在这里添加处理方法
});

datas是一个可迭代的数组或json等即可，第二个参数的一个函数，其中value、key和index分别是当前面对的数据值，数据名和索引。

animation 

$$.animaion(function(deep){
    // 这里编写轮询方法，deep[0-1]，表示动画进度
},duration,function(){
    // 动画结束回调
});

一个状态过渡到另外一个状态，比如位置的缓慢移动，可以调用animation方法，其中duration控制动画时长。
